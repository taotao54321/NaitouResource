駒価値テーブルは 4 種類あり、それぞれ用途が異なる。  
テーブル間で値が異なるのは成駒のみ(ナマ駒の価値は全テーブルで共通)。

| 玉   | 40   |
| 飛   | 17   |
| 角   | 16   |
| 金   | 8    |
| 銀   | 8    |
| 桂   | 4    |
| 香   | 4    |
| 歩   | 1    |
| 龍   | 22   |
| 馬   | 20   |
| 成銀 | 8    |
| 成桂 | 6    |
| 成香 | 5    |
| と金 | 2    |

用途:

* [attacker](@/effect/index.md#attacker) 更新時の比較基準
* 候補手で捕獲する駒の価値算定
